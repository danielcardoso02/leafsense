\section{Test Cases}

\setlength{\parindent}{0pt}
\hspace{1cm}{The reliability of the LeafSense system is verified through a structured series of test cases designed to cover all functional requirements. These tests ensure that the integration between the Raspberry Pi, the sensor array, the execution threads, and the database functions seamlessly over extended periods.}

\setlength{\parindent}{0pt}
\hspace{0cm}{During the implementation phase, several additional test cases were incorporated to verify new features developed for enhanced system functionality. These include:}

\begin{itemize}
    \item \textbf{TCMLAP5 -- Out-of-Distribution Detection:} Verifies that non-plant images are correctly rejected by the ML pipeline using entropy and green-ratio filtering.
    \item \textbf{TCDIS6-TCDIS9:} Disease recommendation generation, multi-class detection, confidence-based alerting, and detailed logging.
    \item \textbf{TCDEF5-TCDEF10:} Deficiency recommendation generation, specific nutrient identification, multi-class detection, alerting, logging, and sensor data correlation.
\end{itemize}

\setlength{\parindent}{0pt}
\hspace{0cm}{The testing phase was divided into the following key categories:}

\begin{itemize}
    \item \textbf{System Setup \& Persistence:} Verification of boot processes, database integrity after reboots, and file storage (Tables \hyperref[tab:setup-tests]{4.3} and \hyperref[tab:data-tests]{4.16}).
    \item \textbf{Data Acquisition:} Validation of sensor readings and scheduled image capture (Tables \hyperref[tab:sensor-tests]{4.4} and \hyperref[tab:image-tests]{4.5}).
    \item \textbf{ML Analysis:} Testing the inference capabilities for disease and deficiency detection models (Tables \hyperref[tab:ml-tests]{4.6} to \hyperref[tab:deficiency-tests]{4.8}).
    \item \textbf{Actuator Control:} Verifying the GPIO logic for the water heater, pH pumps, and nutrient dosing systems (Tables \hyperref[tab:actuator-tests-part1]{4.9} to \hyperref[tab:actuator-tests-part4]{4.12}).
    \item \textbf{GUI \& Configuration:} Checking dashboard responsiveness, alert visualization, and the ability to modify system thresholds (Tables \hyperref[tab:gui-tests-part1]{4.13} to \hyperref[tab:gui-tests-part3]{4.15}).
    \item \textbf{System Integration:} Validating the system's stability during continuous 12-hour and 24-hour operational cycles (\hyperref[tab:data-tests]{Table 4.17}).
\end{itemize}
