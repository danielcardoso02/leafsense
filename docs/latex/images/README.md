# Images Directory

This folder should contain all images/prints for the LaTeX Implementation Report.

## Required Images

See `../README.md` for a complete list of required images with suggested filenames.

## Image Format Recommendations

- **Screenshots**: PNG format, high resolution (at least 1920x1080 for full-screen captures)
- **Photos**: JPEG format, clear lighting, focused
- **Diagrams**: PNG or PDF vector format

## Naming Convention

Use descriptive, lowercase filenames with hyphens:
- `gui-dashboard-light.png`
- `gui-dashboard-dark.png`
- `sensors-connections.jpg`
- `led-gpio-test.jpg`
